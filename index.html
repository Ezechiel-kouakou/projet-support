Hackathon BTS SIO - Référentiel E5

Synthèse du Projet

Ce projet vise à simuler la mise en service d'une agence numérique complète. Nous avons couvert l'ensemble des pôles (Développement, Infrastructure, Support et Pilotage).

Azure
MySQL
GLPI
HTML5/CSS3

Compétences Globales

  • C1 Recensement parc via GLPI Agent.
  • C5 Gestion des sauvegardes et migration vers le Cloud Azure.
  • C12 Évolution agile du site vitrine et intégration d'API.
  • C17 Déploiement et sécurisation via HTTPS (Let's Encrypt).

Réalisation : Développement & Sécurisation

C12, C11, C4, C17 | Membre : FRA

Contexte

Dans le cadre du projet Seyrin, j'ai été chargé de transformer le site vitrine en une plateforme dynamique et sécurisée respectant le RGPD.

Démarche suivie

1. RGPD : Rédaction de la politique de confidentialité (C11).
2. Développement : Création d'API REST pour l'interopérabilité des données (C12).
3. Sécurité : Configuration du nom de domaine et déploiement sur Alwaysdata.

// Exemple API Endpoint
GET /api/v1/seyrin/services

Conclusion : Cette mission m'a permis d'approfondir la gestion des conformités légales et le cycle de vie d'une API.

Réalisation : Infrastructure Cloud & GLPI

C18, C5, C17 | Membre : Rayan

Contexte

Responsable de la mise en place de l'infrastructure de support et de la migration vers le Cloud Azure.

Démarche suivie

1. Azure : Déploiement d'une VM Linux sous Azure (C17).
2. Sauvegarde : Migration des bases de données MySQL vers le nouveau serveur cloud (C5).
3. Support : Gestion des utilisateurs et droits sur l'outil GLPI (C18).

mysqldump -u root -p db_local > backup.sql
mysql -h azure_host -u root -p db_remote < backup.sql

Conclusion : Maîtrise des environnements Cloud public et de la continuité de service lors d'une migration.

Réalisation : Recensement & Analyse d'audience

C1, C11, C17 | Membre : Kylian Gabriel

Contexte

Mise en place du suivi de performance du site et inventaire du parc informatique de l'agence.

Démarche suivie

1. Inventaire : Installation et configuration des agents GLPI sur les postes de l'équipe (C1).
2. Analytics : Intégration du tag Google Analytics pour mesurer la visibilité (C11).
3. Azure : Participation au déploiement des services web (C17).

sudo apt install glpi-agent
glpi-agent --server http://glpi.seyrin.com

Conclusion : J'ai appris à automatiser la remontée d'informations matérielles et à exploiter des indicateurs de visibilité.

Réalisation : Pilotage & Continuité

C12, C4, C5, C17 | Membre : Kouakou

Contexte

En tant que Chef de Projet, j'ai assuré la coordination technique et la pérennité des données.

Démarche suivie

1. Gestion : Création et évolution de la page vitrine responsive (C12).
2. Sécurité : Mise en place de certificats SSL avec Certbot/Let's Encrypt (C17).
3. Fiabilité : Vérification périodique des sauvegardes et tests d'intégrité (C4).

sudo certbot --apache -d seyrinwebsite.com

Conclusion : Cette expérience a renforcé ma capacité à superviser un projet technique complexe de bout en bout.

Réalisation : Support & Maintenance

C1, C12, C17 | Membre : Firadous

Contexte

Soutien au pôle support et participation active à l'amélioration de l'expérience utilisateur du site.

Démarche suivie

1. GLPI : Déploiement des agents et attribution des rôles utilisateurs (C1).
2. Web : Intégration de nouveaux composants UI/UX sur le site (C12).
3. Cloud : Aide au déploiement de l'infrastructure sur Azure (C17).

Conclusion : J'ai pu concilier gestion de parc et développement front-end, deux piliers du support moderne.
